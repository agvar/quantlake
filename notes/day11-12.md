ECS & Fargate: ECS acts as the managing brain (orchestrator) that restarts and monitors your Python container, while Fargate provides the serverless, fully-managed raw hardware (CPU/RAM) required to keep that container's WebSocket alive 24/7.

Egress- means Outbound Internet traffice to download libraries 

`aws ecr create-repository --repository-name quantlake-websocket-relay --image-scanning-configuration scanOnPush=true --encryption-configuration encryptionType=KMS,kmsKey=alias/quantlake-lake --profile quantlake-admin`
  

  this creates a repository in Elastic Container Registry , image-scanning-configuration- check for vulnerabilities in the image when the image is uploaded
  and assigns the KMS CMK key encryption

  `aws ecr get-login-password --region us-east-1 --profile quantlake-admin | docker login --username AWS --password-stdin ${ACCT}.dkr.ecr.us-east-1.amazonaws.com`

this creates a temp password from ECR in AWS and passes it to docker local- this is so the image created in docker local and be moved to the ECR - the default registery for docker is the Docker hub not the AWS ECR, hence the need for the 12 - hour expiring password 

`docker buildx build --platform linux/arm64 -t quantlake-websocket-relay:latest . `
this command uses the latest docker builder toolkit , the -t flag is for the tag- we tag to name the docker image   created
